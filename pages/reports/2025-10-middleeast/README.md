# YECO 香水行业趋势月刊 - v2.0

## 📋 项目说明

这是YECO香水行业趋势10月月刊的2.0版本,采用高端奢侈品杂志风格设计,以黑色为主色调,金色作为点缀。

## 🎨 设计特点

- **配色方案**: 以黑色为主,金色点缀,营造高端奢华感
- **排版风格**: 高端数字杂志,类似Vogue、Wallpaper等奢侈品杂志
- **中东元素**: 融入阿拉伯几何纹样和设计美学
- **响应式设计**: 完美适配桌面、平板和移动设备

## 🎯 图片要求

1. **格式**: JPG或PNG,建议JPG以减小文件大小
2. **质量**: 高清,适合奢侈品杂志标准
3. **风格**:
   - 产品图: 干净背景,专业摄影
   - 氛围图: 电影感,艺术化处理
   - 概念图: 简洁现代,信息图表风格
4. **色调**: 与整体黑金配色协调

## 🚀 使用方法

1. 将图片上传到 `images/` 文件夹
2. 在HTML中替换对应的占位符:
   ```html
   <!-- 原占位符代码 -->
   <div class="image-placeholder">...</div>

   <!-- 替换为 -->
   <img src="images/你的图片.jpg" alt="描述">
   ```
3. 或使用CSS背景图:
   ```css
   .your-element {
       background-image: url('images/你的图片.jpg');
   }
   ```

## 🔧 开发辅助

在浏览器控制台输入以下命令查看所有占位符信息:

```javascript
exportPlaceholderInfo()
```

这将显示一个表格,包含所有图片占位符的位置、描述和建议尺寸。
