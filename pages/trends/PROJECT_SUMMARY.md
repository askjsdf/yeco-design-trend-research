## 使用说明

### 快速启动
```bash
cd /Volumes/SD2/yeco
python -m http.server 8000
# 访问: http://localhost:8000/pages/trends/
```

### 添加报告
编辑 `data/reports-index.json`，添加新报告对象。

### 修改样式
编辑 `style.css` 或全局 `variables.css`。

### 添加翻译
编辑 `i18n/` 目录下的对应语言文件。
